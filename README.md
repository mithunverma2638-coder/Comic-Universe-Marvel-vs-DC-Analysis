# Comics Universe — Marvel vs DC Analysis

![Dashboard Preview](3416207_59271.jpg)

## Project Overview
Complete data analysis project on 23,272 Marvel and DC comic characters
using Power BI, SQL, and HTML dashboard.

## Dataset
- Source: FiveThirtyEight
- Records: 23,272 characters
- Publisher: Marvel (16,376) | DC (6,896)

## Tools Used
| Tool | Purpose |
|------|---------|
| Power BI | Interactive Dashboard |
| SQL | Data Analysis Queries |
| HTML/CSS | Web Dashboard |
| CSV | Raw Dataset |

## Dashboard Pages
1. Overview — KPI cards + publisher split
2. Marvel vs DC Battle — Head-to-head comparison
3. Characters Deep Dive — Gender, alignment, status
4. Timeline — Characters introduced per year
5. Top Heroes — Top 10 by appearances

## Key Insights
- Spider-Man is #1 with 4,000+ appearances
- Marvel has 70.37% of all characters
- 76.52% characters are Living
- Good Characters outnumber Bad Characters

## Files
- `Comic_Universe_Marvel_vs_DC_Analysis.pbix` — Power BI Dashboard
- `marvel_dc_analysis.sql` — SQL queries (10 analysis sections)
- `Marvel_vs_DC_Analysis.html` — Web Dashboard
- `Comic_Universe_Marvel_vs_DC_Analysis.csv` — Dataset
