# 🦸 Comics Universe — Marvel vs DC Analysis

## 📌 Project Overview
Complete end-to-end data analysis project on **23,272 Marvel and DC comic characters**
built using Power BI, SQL, and HTML dashboard.

---

## 📊 Dataset
| Detail | Info |
|--------|------|
| Source | FiveThirtyEight |
| Total Records | 23,272 characters |
| Marvel | 16,376 (70.37%) |
| DC | 6,896 (29.63%) |
| Year Span | 1935 — 2013 |

---

---

## 🔍 Key Insights
- 🕷️ **Spider-Man** is #1 with **4,000+ appearances**
- 🦇 **Batman** leads DC with **3,100+ appearances**
- 📈 Marvel has **2.4x more characters** than DC
- ✅ **76.52%** characters are Living
- 💪 **Good Characters** outnumber Bad Characters in both publishers
- 👨 Male characters dominate — **70%+** in both Marvel and DC

---

## 🗄️ SQL Analysis Covers
1. Publisher breakdown
2. Alignment analysis (Good/Bad/Neutral)
3. Gender & diversity
4. Mortality/survival status
5. Appearances & popularity metrics
6. Chronological trends (by decade)
7. Comic age breakdown (Golden/Silver/Bronze/Modern)
8. Identity type (Secret vs Public)
9. Physical traits (eye & hair color)
10. Villain & female hero profiles
